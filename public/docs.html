🚀 Quick Start Guide

Get up and running with HypeAI in under 5 minutes

1. Install Dependencies

npm install @hypeai/sdk ethers

2. Initialize SDK

import { HypeAI } from '@hypeai/sdk';

const hypeai = new HypeAI({
  apiKey: 'your-api-key',
  network: 'polygon'
});

3. Start Trading

const prediction = await hypeai.ai.getPrediction('BTC/USD');
console.log(`AI Prediction: ${prediction.price} (${prediction.confidence}% confidence)`);
