Authentication

HypeAI uses API keys for authentication. You can generate an API key from your dashboard.

// JavaScript Example const HypeAI = require('@hypeai/sdk'); const client = new HypeAI({ apiKey: 'your_api_key_here', network: 'bsc-mainnet' }); // Verify connection await client.ping();

AI Agents API

GET /api/v1/agents

Get list of all available AI agents.

GET /api/v1/agents/{agentId}/analyze

Request analysis from a specific AI agent.

Request Parameters

Parameter Type Required Description
agentId string Yes Unique identifier for the AI agent
symbol string Yes Trading pair symbol (e.g., BNB/USDT)
timeframe string No Analysis timeframe (1h, 4h, 1d)
indicators array No Technical indicators to include

Example Response

{ "agentId": "market-analyzer", "symbol": "BNB/USDT", "timestamp": 1706198400, "analysis": { "trend": "bullish", "confidence": 0.87, "signal": "BUY", "priceTarget": 520.00, "stopLoss": 480.00 }, "indicators": { "RSI": 62.4, "MACD": "bullish", "volume": "increasing" } }

Trading API

POST /api/v1/orders

Place a new trading order.

// Place Market Order const order = await client.trading.createOrder({ symbol: 'BNB/USDT', side: 'buy', type: 'market', quantity: 10 }); console.log(order.orderId);

JavaScript SDK

Install the official HypeAI JavaScript SDK via npm:

npm install @hypeai/sdk # or with yarn yarn add @hypeai/sdk
import { HypeAI } from '@hypeai/sdk'; const hypeai = new HypeAI({ apiKey: process.env.HYPEAI_API_KEY }); // Get AI analysis const analysis = await hypeai.agents.analyze('market-analyzer', { symbol: 'BNB/USDT', timeframe: '1h' }); console.log(analysis.signal); // BUY, SELL, or HOLD
