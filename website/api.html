API Endpoints

💰 Tokens API

GET /v1/tokens

Get list of all supported tokens with current prices and 24h changes.

{ "tokens": [ { "symbol": "HYPEAI", "name": "HypeAI Token", "price": 0.001, "change24h": 12.5, "volume24h": 1250000, "marketCap": 1200000 } ] }
GET /v1/tokens/{symbol}

Get detailed information about a specific token.

Parameter Type Description
symbol string Token symbol (e.g., HYPEAI, BTC, ETH)
GET /v1/tokens/{symbol}/history

Get historical price data for a token with customizable timeframe.

Parameter Type Description
symbol string Token symbol
interval string 1m, 5m, 15m, 1h, 4h, 1d, 1w
limit number Number of data points (max 1000)

🔒 Staking API

GET /v1/staking/pools

Get all available staking pools with current APY rates.

POST /v1/staking/stake

Create a new staking position.

{ "amount": "1000", "lockPeriod": 365, "walletAddress": "0x..." }
GET /v1/staking/positions/{address}

Get all staking positions for a wallet address.

POST /v1/staking/unstake

Unstake tokens after lock period expires.

🗳️ Governance API

GET /v1/governance/proposals

Get all governance proposals with voting status.

POST /v1/governance/vote

Cast a vote on an active proposal.

{ "proposalId": 123, "vote": "FOR", "votingPower": 1000, "walletAddress": "0x..." }
POST /v1/governance/propose

Create a new governance proposal (requires minimum token holdings).

🤖 AI Predictions API

GET /v1/ai/predict/{pair}

Get AI-powered price prediction for a trading pair.

{ "pair": "BTC/USD", "currentPrice": 45000, "prediction": { "price": 47500, "confidence": 87.5, "timeframe": "24h", "trend": "BULLISH" }, "signals": ["STRONG_BUY", "VOLUME_SURGE", "POSITIVE_SENTIMENT"] }
GET /v1/ai/sentiment/{symbol}

Get market sentiment analysis for a token based on social media and news.

📊 Analytics API

GET /v1/analytics/portfolio/{address}

Get comprehensive portfolio analytics for a wallet address.

GET /v1/analytics/market-overview

Get overall market statistics and trends.

GET /v1/analytics/leaderboard

Get top traders and stakers leaderboard.

🔌 WebSocket API

Real-time Data Streaming

Connect to our WebSocket endpoint for real-time price updates, trade executions, and AI predictions.

// WebSocket Connection const ws = new WebSocket('wss://api.hypeai.io/v1/ws'); ws.onopen = () => { ws.send(JSON.stringify({ type: 'subscribe', channels: ['prices', 'predictions', 'trades'] })); }; ws.onmessage = (event) => { const data = JSON.parse(event.data); console.log('Real-time update:', data); };
