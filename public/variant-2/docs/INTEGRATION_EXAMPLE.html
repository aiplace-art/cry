Integration Instructions

Follow these steps to add mobile optimizations to your pages:

Step 1: Add CSS (in <head>)

<!-- Mobile Optimizations CSS (MUST BE LAST) --> <link rel="stylesheet" href="css/mobile-optimizations.css">

Step 2: Add JavaScript (before </body>)

<!-- Mobile Navigation JS --> <script src="js/mobile-nav.js"></script>

Step 3: Test! No HTML changes needed

The hamburger menu is automatically generated. Just add the CSS and JS files!
