## üìù Writing Rules Cheat Sheet

### ALWAYS:
‚úÖ Use precise numbers ("85%", "1B tokens", "62% APY")
‚úÖ Cite sources and proof links
‚úÖ Disclose risks honestly
‚úÖ Use technical specifics ("LSTM", "Polygon", "Smart Contracts")
‚úÖ Say "Projected" or "Estimated" for future claims
‚úÖ Include "Not financial advice"
‚úÖ Link to verification (blockchain explorers, GitHub, audits)

### NEVER:
‚ùå Promise wealth ("millionaire", "get rich", "financial freedom")
‚ùå Use vague hope words ("opportunity", "future", "success")
‚ùå Make guarantees ("guaranteed profits", "risk-free", "100% safe")
‚ùå Create urgency ("limited time", "don't miss out", "FOMO")
‚ùå Hide risks or limitations
‚ùå Use unverifiable superlatives ("best", "#1", "revolutionary")
‚ùå Present projections as facts

## üéØ The Ultimate Test

**Before publishing ANY content, ask:**

1. **Data Test**: "Can I prove this with numbers/links?"
   - If NO ‚Üí Rewrite or remove

2. **Risk Test**: "Am I hiding any downsides?"
   - If YES ‚Üí Add disclaimers

3. **Hope Test**: "Does this rely on emotion or facts?"
   - If emotion ‚Üí Rewrite with facts

4. **Russian Test**: "Would this translate to '–ù–µ –Ω–∞–¥–µ–∂–¥–∞. –ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞.'?"
   - If NO ‚Üí Doesn't match our brand

5. **Legal Test**: "Could SEC/regulators call this misleading?"
   - If MAYBE ‚Üí Add disclaimers or rewrite
