Authentication

All API requests require authentication using an API key. Include your key in the Authorization header.

POST /api/v1/auth/login

Authenticate with your credentials to receive an API key and access token.

Request Body

Parameter Type Required Description
email string Required User email address
password string Required User password
Request Example 
curl -X POST https://api.hypeai.io/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "your_secure_password"
  }'
Response Example 
{
  "success": true,
  "data": {
    "apiKey": "hypeai_1234567890abcdef",
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expiresIn": 86400,
    "user": {
      "id": "usr_abc123",
      "email": "user@example.com",
      "tier": "pro"
    }
  }
}

POST /api/v1/agents/{agentId}/execute

Execute a specific AI agent with custom parameters and receive analysis results.

Request Body

Parameter Type Required Description
task string Required Task type: "analyze", "predict", "execute"
parameters object Required Task-specific parameters
async boolean Optional Execute asynchronously (default: false)
Request Example 
curl -X POST https://api.hypeai.io/v1/agents/agent_market_analyzer/execute \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "task": "analyze",
    "parameters": {
      "symbol": "BNB/USDT",
      "timeframe": "1h",
      "indicators": ["RSI", "MACD", "EMA"]
    }
  }'

Trading

Execute trades, manage orders, and access real-time market data on Binance Smart Chain.

POST /api/v1/trading/orders

Place a new trading order (market, limit, or stop-loss).

Request Body

Parameter Type Required Description
symbol string Required Trading pair (e.g., "HYPE/BNB")
type string Required Order type: "market", "limit", "stop_loss"
side string Required "buy" or "sell"
amount number Required Order amount in base currency
price number Optional Limit price (required for limit orders)
Request Example 
curl -X POST https://api.hypeai.io/v1/trading/orders \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "symbol": "HYPE/BNB",
    "type": "limit",
    "side": "buy",
    "amount": 1000,
    "price": 0.001
  }'

WebSocket API

Connect to real-time data streams for prices, trades, and agent updates.

WS wss://ws.hypeai.io/v1/stream

Connect to WebSocket server and subscribe to real-time channels.

JavaScript Example 
const ws = new WebSocket('wss://ws.hypeai.io/v1/stream');

ws.onopen = () => {
  // Subscribe to price updates
  ws.send(JSON.stringify({
    action: 'subscribe',
    channels: ['prices:HYPE/BNB', 'agents:status']
  }));
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Received:', data);
};

// Example message
{
  "channel": "prices:HYPE/BNB",
  "data": {
    "price": 0.001,
    "timestamp": "2025-10-19T17:00:00Z"
  }
}
